# black hole
Spatial storage for Stardust XR, like windows peek but in 3D
> [!IMPORTANT]  
> Requires the [Stardust XR Server](https://github.com/StardustXR/server) to be running.

If you installed the Stardust XR server via:  
```note
sudo dnf group install stardust-xr
```
Or if you installed via the [installation script](https://github.com/cyberneticmelon/usefulscripts/blob/main/stardustxr_setup.sh), Black Hole comes pre-installed

## How To Use
Launch with `black-hole`.

Touching or clicking the minus sign will hide all clients within it's radius. 
Clicking it again will replace them to their original location.
